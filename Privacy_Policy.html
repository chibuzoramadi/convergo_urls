Convergo Privacy Policy

Last Updated: December 9, 2025

1. Who We Are

Convergo is a SaaS platform that enables small businesses to connect their Meta WhatsApp Business accounts (with future support for Instagram and Facebook Messenger) to OpenAI-powered AI assistants. Our platform provides automated reply capabilities, conversation analytics, and a centralized dashboard for message management. [memory:6][web:40]

We operate as a multi-tenant service hosted on AWS infrastructure, using MySQL databases for data storage and JWT-based authentication for account security. Our service processes messages through Meta's WhatsApp Business API webhooks and generates AI responses via OpenAI's API. [memory:5][web:17]

2. Data We Collect

Convergo collects the following categories of personal data to deliver our core services:

Account Data

  • Email address, full name, business name, and phone number (collected during signup)
  • Password (hashed with bcrypt before storage)
  • Subscription plan and billing details (processed via third-party payment providers)

Meta Integration Data

  • WhatsApp Business Account (WABA) ID and phone numbers linked via OAuth
  • Incoming/outgoing WhatsApp conversation messages received via webhooks
  • Message metadata (timestamps, sender details, message status)

AI Processing Data

  • Conversation prompts sent to OpenAI API
  • AI-generated reply content
  • AI configuration settings (reply templates, tone preferences)

Technical Data

  • IP addresses, browser type, device information, and session data
  • Usage analytics (dashboard views, API call frequency)
  • Server logs for security monitoring

3. How We Use Your Data

User data powers Convergo's core functionality:

  • Service Delivery: Generate AI replies, display dashboard analytics
  • Meta Integration: Forward webhook messages to OpenAI, send AI replies back through WhatsApp API
  • Account Management: Authentication, subscription status, usage limits
  • Improvement: Anonymized analytics for product development

4. Data Sharing and Third Parties

Convergo shares data only with service providers essential to operations:

  • Meta: WhatsApp Business API for message delivery and OAuth authentication [web:17]
  • OpenAI: ChatGPT API for generating AI replies (data processed per their DPA) [web:20]
  • AWS: EC2 hosting, RDS MySQL, encryption services (AWS Privacy Reference Architecture compliant) [web:19]
  • Payment Processors: Stripe/Paddle for subscriptions (no card data stored by Convergo)

No data sales: Convergo never sells user data to advertisers or third parties.

5. Data Security

  • Encryption: HTTPS/TLS for transit, AES-256 at rest (AWS-managed keys)
  • Authentication: JWT tokens with 30-minute expiry, bcrypt password hashing
  • Isolation: Multi-tenant MySQL schemas separate customer data
  • Access: Least-privilege IAM roles, audit logging, regular penetration testing

6. Data Retention

  • Active Accounts: Data retained while subscription active
  • Inactive Accounts: Auto-Freeze after 90 days of no login/subscription
  • Server Logs: 30 days maximum (anonymized after 7 days)
  • Backups: Encrypted, retained 7 days, purged on account deletion

7. Your Rights

Users have full control over their data:

  • Access account data via dashboard
  • Download conversation export (CSV/JSON)
  • Delete account and all data instantly (/account → Delete Account)
  • GDPR/CCPA rights: contact support@convergo.com
